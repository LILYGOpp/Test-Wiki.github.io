## Quick Start

1. **Choose a Product** - Select a product you're interested in from the categories above
2. **Read Documentation** - Review detailed user guides and hardware specifications
3. **Download Resources** - Get drivers, sample code, and firmware
4. **Start Development** - Follow the documentation for hardware connections and software programming
5. **Contact Support** - Reach out to our technical support if you have questions

## Documentation Structure

Each product's documentation typically includes:

- **Product Overview** - Specifications, features, and application scenarios
- **Quick Start** - Basic usage steps and getting started guide
- **Hardware Connection** - Wiring diagrams, pin definitions, and hardware instructions
- **Software Development** - Programming guides, API documentation, and sample code
- **FAQ** - Frequently asked questions and troubleshooting
- **Related Resources** - Download links and reference materials

## Development Framework Support

LILYGO products support multiple mainstream development frameworks:

- **Arduino** - Easy to use, suitable for rapid prototyping
- **ESP-IDF** - Official development framework with complete functionality
- **MicroPython** - Scripted development for quick start
- **PlatformIO** - Integrated development environment supporting multiple platforms
