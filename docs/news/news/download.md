# 资源下载

本页面提供 LILYGO 相关的所有资源下载。

## 硬件资源

### 原理图和 PCB 设计

所有产品的原理图、PCB 文件和设计资料可在 [GitHub](https://github.com/Xinyuan-LilyGO) 上找到。

### 数据手册

- ESP32 相关芯片数据手册
- 外围芯片数据手册（显示屏、传感器等）
- 接口规范

## 软件工具

### 开发工具

- **Arduino IDE** - [官方下载](https://www.arduino.cc/en/software)
- **VS Code** - [官方下载](https://code.visualstudio.com/)
- **PlatformIO** - VS Code 扩展，支持多种开发板

### 驱动程序

- **CH340/CH341** - USB 转串口芯片驱动
- **CP2102** - 另一种 USB 转串口驱动
- **FTDI** - FTDI 芯片驱动

### 烧录工具

- **ESPFlash** - 用户友好的 ESP32 烧录工具
- **esptool.py** - 命令行烧录工具
- **Flash Download Tool** - 官方烧录工具
